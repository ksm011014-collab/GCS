backend 파일

backend/drone_specs.py
DJI 기체별 사양 데이터
최대 수평 속도, 상승/하강 속도, 최대 고도, 비행 시간, 무게, RCS 추정값을 제공

backend/formation.py
다중 드론 편대 비행에서 첫 웨이포인트 동기화를 담당
첫 웨이포인트에서 대기 후 동시 출발(군집 시나리오 용)

backend/logger.py
완료된 시나리오의 텔레메트리 샘플을 CSV로 저장하는 로직
async API 형태를 유지하지만 내부 파일 쓰기는 단순 CSV 저장입니다.

backend/main.py
백엔드 핵심 엔트리포인트
FastAPI 앱, REST API, WebSocket, 시뮬레이션 루프, 임무 상태 전환, 텔레메트리 송신, 진행률 계산, 실기체 MAVLink WebSocket 등 포함

backend/mavlink_bridge.py
Pixhawk/MAVLink 연결 보조 코드
UDP/Serial 연결 생성, MAVLink 메시지 필터링, 위치/속도/자세 데이터 추출을 담당

backend/physics.py
순수 물리 계산 모듈
RK4 방식으로 드론 상태를 한 스텝 전진시키고, 속도/가속도/위치 텔레메트리를 계산

backend/scenario.py
비행 시나리오 정의, 저장, 로드, 삭제, 수정 로직
기본 시나리오와 사용자 저장 JSON 시나리오를 ScenarioDefinition으로 변환

backend/ws_manager.py
WebSocket 연결 관리 클래스
연결 등록, 해제, 개별 JSON 송신, 브로드캐스트 기능을 제공

backend/__init__.py
backend 폴더를 Python 패키지로 인식시키는 파일
backend.main처럼 import가능하게


frontend 주요 파일

frontend/index.html
웹 UI


frontend/css/01_main.css
CSS 묶음

frontend/css/basic.css
기본 색상, 전역 스타일, 테마 관련 기본값을 담당

frontend/css/components.css
공통 버튼, 셀렉트, 입력창, 카드성 UI 요소 스타일

frontend/css/right.css
오른쪽 패널, 탭, 시나리오 빌더, 실기체/비행 패널 스타일

frontend/css/bottom.css
하단 진행률 바와 시작/정지/초기화/저장 버튼 영역 스타일

frontend/css/mobile.css
모바일 화면 대응 스타일

frontend/js/main.js
프론트엔드 전체 조정자
드론/시나리오 선택, 시뮬레이션 시작/정지/재개, 실기체 연결, 패널 전환, 저장, 각 모듈 초기화를 담당

frontend/js/socket.js
WebSocket 진입점

frontend/js/map.js
Leaflet 지도 전담 모듈
지도 초기화, 드론 마커, 실제/계획 경로, 홈/웨이포인트 표시, 시나리오 편집용 지도 동작, ADI 표시를 관리

frontend/js/telemetry.js
텔레메트리 카드 UI를 관리
속도, 고도, 위치, 가속도 등 수신 데이터를 화면에 반영

frontend/js/log.js
화면 로그 패널을 관리
WebSocket 로그 메시지나 UI 액션 로그를 append-only 방식으로 표시

frontend/js/flight_log.js
브라우저 측 비행 데이터 기록/다운로드 기능
텔레메트리 이벤트를 모아 CSV로 저장할 때 사용

frontend/js/scenario_builder.js
사용자 비행 계획 생성/수정/삭제 UI 로직
지도 편집 상태와 입력 폼을 동기화하고 백엔드 custom scenario API를 호출

frontend/assets/drone-icons
